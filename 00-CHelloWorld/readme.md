# Resultados del TP0

* **Compilador seleccionado:** `gcc` (MSYS2)
* **Versión del compilador:** 15.2.0
* **Versión de C soportada:** C90/89, C99, C11, C17, C23, y experimentalmente C2Y.

* **Bibliografía:** https://gcc.gnu.org/onlinedocs/gcc-15.2.0/gcc/Standards.html#C-Language

* **Para compilar con make:** Usar el comando `make` 
* **Para ejecutar con make:** Usar el comando `make run`
